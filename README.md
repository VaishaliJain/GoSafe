# NewsScapper
Run java TheHinduScraper to fetch crime locations for Bangalore and Delhi. 
Requirements:
	<br/>* Stanford NLP CRF Classifiers: english.all.3class.distsim.crf.ser.gz [Version 3.9.1].
	<br/>* JSoup 1.11.2
	<br/>* Java 1.8 or later.
